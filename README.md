# Krish Rathi Portfolio

Static Vite portfolio for Krish Rathi, built from the comic-style HTML theme in `index.html`.

## What It Shows

- AI/ML and full-stack profile
- Featured projects
- Current GitHub and LeetCode stats
- Proof Wall and interactive Mission Board
- Resume download from `public/Krish_Rathi_Resume.pdf`

## SEO Assets

- Canonical, Open Graph, Twitter card, robots, and sitemap metadata
- JSON-LD for `Person`, `ProfilePage`, `WebSite`, `ItemList`, `FAQPage`, and `BreadcrumbList`
- Social preview image at `public/og-image.svg`
- Crawler summary at `public/llms.txt`

## Local Development

```bash
npm install
npm run dev
```

## Checks

```bash
npm run lint
npm run build
npm audit --audit-level=moderate
```
